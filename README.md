# TrackMoney

A simple personal website to track your earnings and spendings.

## Features

- Add earning or spending entries with a note and amount.
- View total balance, total earnings, and total spendings.
- See a timestamped history of all entries.
- Delete individual entries or clear all data.
- Data persists in your browser using `localStorage`.

## Run locally

Since this is a static app, you can open `index.html` directly, or run:

```bash
python3 -m http.server 4173
```

Then visit <http://localhost:4173>.
